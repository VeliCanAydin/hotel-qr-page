# Allergen Icons

Bu klasör, siteye bağlamadan önce ikonları görsel olarak kontrol etmeniz için oluşturuldu.

Dosya eşleşmeleri:
- `nuts.svg` -> nuts
- `peanut.svg` -> peanut
- `gluten.svg` -> gluten
- `dairy.svg` -> dairy
- `shellfish.svg` -> shellfish
- `soy.svg` -> soy
- `egg.svg` -> egg
- `sesame.svg` -> sesame
- `fish.svg` -> fish

Şu an ikonlar özgün SVG tasarımlardır. Beğendiğiniz seti seçtikten sonra bunları siteye bağlarız.
